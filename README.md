 VM Health Check

## Overview

VM Health Check is a Bash script designed to monitor the basic health of a Linux virtual machine.

The script checks CPU utilization, memory utilization, and disk space utilization and reports the health status of the system based on a predefined threshold.

## Features

- Checks CPU utilization
- Checks memory usage
- Checks disk space utilization
- Uses a configurable utilization threshold
- Provides color-coded terminal output
- Provides an explanation mode for more detailed results

## Requirements

- Linux-based virtual machine or server
- Bash shell
- Terminal access

## Usage

Make the script executable:

chmod +x vm_health_check.sh
